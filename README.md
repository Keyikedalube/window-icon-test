# window-icon-test
A test project for fixing synfig issue https://github.com/synfig/synfig/issues/1412

# How to build
```
git clone https://github.com/Keyikedalube/window-icon-test.git
cd window-icon-test

mkdir build-dir
cd build-dir

cmake ..
make

./window-icon-test
```
